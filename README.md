# Simulate Long Horizon Code Generation

This example simulates the concept of a "long-horizon coding LLM" by demonstrating how a complex task can be broken down into a plan, individual code modules generated for each part, and then integrated into a single, functional program. It illustrates the LLM's ability to understand overall project architecture and dependencies, rather than just generating isolated code snippets.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run the simulation: `python main.py`
3. This will create a new file `generated_program.py`. Run the generated program with various inputs: `python generated_program.py 5`, `python generated_program.py -3`, `python generated_program.py hello`
